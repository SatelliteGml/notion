Техническое задание: Система управления заметками
1. Введение
Разработать простую систему управления заметками,
которая позволит пользователям добавлять, удалять и просматривать заметки.
Все заметки должны храниться в текстовом файле.

2. Цели и задачи.
Реализовать возможность добавления новых заметок.
Реализовать возможность удаления существующих заметок по индексу.
Реализовать возможность просмотра всех заметок.
Все данные должны сохраняться в текстовом файле notes.txt.
Обрабатывать возможные исключения, связанные с работой с файлами.
3. Требования к функциональности
Добавление заметки

Пользователь вводит текст заметки.
Программа добавляет заметку в конец файла notes.txt.
Если файл не существует, он должен быть создан.
Удаление заметки

Пользователь вводит индекс заметки (начиная с 1).
Программа удаляет заметку с указанным индексом.
Если индекс выходит за пределы, программа должна выводить сообщение об ошибке.
После удаления заметки остальные заметки должны сдвигаться на одну позицию вверх.
Просмотр заметок

Программа читает все заметки из файла notes.txt и выводит их на экран с указанием индекса.
4. Структура проекта
Классы
NoteManager — основной класс, отвечающий за управление заметками.
Note — класс, представляющий заметку (может содержать дополнительные атрибуты, если нужно).
5. Обработка исключений.
Обработать исключения, связанные с:
Неправильным вводом данных (например, не числовой индекс).
Проблемами с доступом к файлу (например, отсутствие прав на запись/чтение).
Ошибками при работе с файлами (например, файл не найден).
6. Технические детали
Язык программирования: Java
Версия JDK: 8 и выше
7. Тестирование
Провести тестирование всех функций.
Проверить обработку исключений в различных сценариях.
Примерный алгоритм работы программы
Запуск программы.
Выбор действия (добавить заметку, удалить заметку, просмотреть заметки).
Выполнение выбранного действия.
Повторение процесса до выбора выхода.
